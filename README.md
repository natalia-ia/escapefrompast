Jogo com IA - Chronos Escape de Computação
Este projeto foi desenvolvido como parte de um trabalho acadêmico.

O projeto é um jogo com o nome “Escape.from_past()” em que um personagem escolhido no menu se encontrará em uma situação inusitada: uma viagem no tempo. O objetivo do jogador é resolver enigmas, desafios de lógica e puzzles que estarão relacionados a cada marco temporal da história da computação a fim de retornar ao presente. 
O mecanismo do jogo se baseia em fases com sistemas de progressão em que o jogador deverá realizar dinâmicas para desbloqueá-las em um determinado tempo, acumulando pontos. Caso não consiga chegar a uma saída, o jogador pode utilizar dicas como assistente virtual para auxiliá-lo a desvendar os desafios.  

 Principais Funcionalidades:

- Sistema de fases progressivas
- Puzzles baseados em eventos históricos
- Sistema de tempo e pontuação
- Inventário de itens

 Tecnologias:
- Python
- Pygame
- Ollama

 Instruções de Instalação:
1- Clonar o repositório:
git clone https://github.com/natalia-ia/escapefrompast.git
2- Instalar as dependências:
cd escapefrompast/Pygamepip install -r requirements.tx
ollama pull qwen2.5:1.5b
3- Executar o jogo:
cd menupython jogo.py
 

O diferencial do “Escape.from_past()” é que ele caracteriza-se por ser um jogo capaz de unir o desenvolvimento de raciocínio lógico do jogador com conhecimentos fundamentais da história computacional, tornando-se educativo e interativo em sua essência.
